=======
Fklearn
=======

**fklearn** uses functional programming principles to make it easier to solve real business problems using Machine Learning.

The name is a reference to the widely known `scikit-learn <https://scikit-learn.org/stable/>`_ library.

**fklearn Principles**

#. Validation should reflect real-life situations.
#. Production models should match validated models.
#. Models should be production-ready with few extra steps.
#. Reproducibility and in-depth analysis of model results should be easy to achieve.
